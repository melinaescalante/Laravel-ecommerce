# Ecommerce - Laravel

ABM website for video game sales built with Laravel and the Blade template engine.

## FUNCTIONALITIES
- ABM games
- Add to cart
- Delete from cart
- Payment plataform
- Search of games
- Filter by clasification
- View of each game
- Register and authentication
- Profile with purchases and details.
- Age restriction of games

## Running project
1. Clone project on folder xampp/ htdocs
2. Config env
3. Run composer
``` bash
composer i 
```
4. Run npm
``` bash
npm i
```
5. Run command
 ``` bash
php artisan key generate
```
6. Run migration
 ``` bash
php artisan migrate:fresh
```
7. Run seeders
 ``` bash
php artisan db:seed
```
8. Make link
 ``` bash
php artisan storage:link
```
